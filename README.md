# cmd-line-dict-tool
Command line dictonary tool for words.

## Installation Steps
`npm install`

## How to use?
### 1. Word Definitions
`node ./dict defn <word>`

### 2. Word Synonyms
`node ./dict syn <word>`

### 3. Word Antonyms
`node ./dict ant <word>`

### 4. Word Examples
`node ./dict ex <word>`

### 5. Word Full Dict
`node ./dict <word>`

### 6. Word of the Day Full Dict
`node ./dict`

### 7. Word Game
`node ./dict play`
